# React Components

App on Vercel:  
https://react-components-n77e1qy1m-yoshino1.vercel.app

## Start at Local

```
docker compose up
```

Next.js works on 4000 port.  
Storybook works on 6006 port.

## Lint and Format

Lint for Next.js

```
docker compose run yarn lint
```

Format for Next.js

```
docker compose run yarn format
```
